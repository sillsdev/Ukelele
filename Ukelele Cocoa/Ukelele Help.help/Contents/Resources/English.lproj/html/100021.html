In the Keyboard menu is an item Edit Key… This brings up a dialog where you can specify the key by its key code, and also give the modifiers, then enter new output. If there is a selected key when you choose this command, that key’s key code is entered, as well as whatever modifiers are active when you choose the menu item.

With this item, it is also possible to make the key into a dead key. See the discussion elsewhere on dead keys, but simply here you can choose the dead key state and (optionally) the terminator.
