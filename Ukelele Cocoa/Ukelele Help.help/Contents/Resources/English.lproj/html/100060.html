Setting the language for your keyboard layout is simple, but it can only be done if it is part of a keyboard layout collection (also known as a bundle). If you are working with an unbundled keyboard layout, the menu items for setting a language will be unavailable.

To convert to a collection, you can create a new keyboard layout collection, and then use the + button to add your existing keyboard layout to it. You can even add it by using the keyboard window.
