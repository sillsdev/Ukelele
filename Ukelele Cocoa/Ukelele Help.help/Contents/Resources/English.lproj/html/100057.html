Basically, you don’t. Changing the modifier keys (shift, option, command, control, caps lock) can’t be done with Ukelele.

If you want to rearrange the modifier keys, System Preferences lets you do that.

If you really want to change a modifier key into something else, consider an alternative solution, such as Better Touch Tool, Karabiner, or DoubleCommand.
