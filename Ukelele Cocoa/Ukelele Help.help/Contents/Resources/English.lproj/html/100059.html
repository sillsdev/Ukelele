There are several things involved here, but I will only deal with the core issue: getting a file that Ukelele can use for a flag icon.

The flag icon (what appears in the menu bar for the input menu) is a specialised file format, which requires you to take whatever graphics format you have (JPEG, GIF, PNG, TIFF, etc) and convert it to the correct format, known as icns. You cannot just change the file extension and make it work! You need to use one of a number of tools that are available for the job.

Here are some options:
Image2icon (free on Mac App Store)
iConvertIcons (free web application or non-free app on Mac App Store)
FastIcns (free Mac app)
GraphicConverter (Shareware that does almost anything you need with graphics files)
Icon Slate (non-free app on Mac App Store)

Adding the icon to your keyboard layout is done either manually (if you are using a stand-alone keyboard layout) or in Ukelele (if you are using a keyboard layout collection).
