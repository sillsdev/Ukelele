The modifier keys that Mac OS X uses are:

shift — ⇧
caps lock — ⇪
option — ⌥
command — ⌘
control — ⌃

On some keyboards, there are other keys which function somewhat like modifiers, such as fn, but these are not true modifier keys. Only these five modifier keys are recognised as modifiers by the operating system.

There are left and right modifier keys for shift, option and control, but these are of extremely limited usefulness. No Apple keyboard currently in use can produce a right shift distinct from a left shift — both shift keys are seen as left shift keys. For this reason, Ukelele can simplify modifier sets from a keyboard layout by removing references to the right modifier keys. This makes the modifiers table easier to understand.
