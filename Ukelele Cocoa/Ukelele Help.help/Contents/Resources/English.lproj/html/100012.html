There are six standard keyboard layouts:
  • Empty keyboard layout (no output except the basics)
  • QWERTY (standard English keyboard layout)
  • QWERTZ (standard German keyboard layout)
  • AZERTY (standard French keyboard layout)
  • Dvorak (ergonomic English keyboard layout)
  • Colemak (more recent ergonomic English keyboard layout)
By choosing Standard Keyboard Layoutâ€¦ from the + button below the list of keyboard layouts in the collection, you will get a dialog asking you for a name for the keyboard layout, and asking you to specify up to three standard keyboard layouts.

One keyboard layout will be the base layout, that is, the layout for when no modifier keys are pressed. You can then optionally add one or two more keyboard layouts when the command or caps lock key is pressed. This is often used to ensure that keyboard shortcuts with the command key are handled correctly by the system and are easy to remember.

For each type of keyboard layout, a lowercase and an uppercase variation is generated, so that no modifiers gives lowercase, and shift gives uppercase.
