First, ensure that you have set a name for the keyboard layout (Keyboard menu, Set Keyboard Name and Script). This is the name that will appear in the input menu.


Next, save the keyboard layout to a work folder. It doesn’t matter where at this point.

Next, open the Organiser. If the “Not Installed” folder shown on the left is different to where you saved the keyboard layout, click Set Folder and choose the folder.

Drag the keyboard layout into the appropriate “Installed” folder, usually the one for the current user.


It is best to log out (after quitting Ukelele) and logging in again, to ensure that the system sees your keyboard layout.

Open System Preferences, and choose the Keyboard pane, then the Input Sources tab. Click the + button on the left, and select your keyboard layout. Typing the first few letters of the name will help you find it. Click Add, and you are done.
